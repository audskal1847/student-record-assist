# 학생부 입력 어시스트

학생들의 활동 키워드만 입력하면, AI가 가이드북·엑셀 사전 기준에 맞는 풍성한 학생부 문장을 자동으로 생성해 주는 도구입니다.

## 주요 기능
- 학생별 진로 맞춤형 학생부 문장 생성
- 학생부 작성 가이드북 및 엑셀 사전 기반
- 나이스 한도(1500바이트) 이내 자동 분량 조절
- 마크다운, 과목명, '학생은' 표현, 구체적 수치 자동 제거

## 사용 방법
1. 왼쪽 사이드바에 Google AI API 키 입력
   - 무료 발급: https://aistudio.google.com/app/apikey
2. 과목, 프로젝트명, 진학 희망 학과 입력
3. 학생 활동 키워드 입력
4. '맞춤형 문장 생성' 버튼 클릭

## 주의사항
- AI 생성 결과는 반드시 교사가 확인 후 사용하세요.
- 검증되지 않은 수치는 자동 제거되지만, 사실 관계는 직접 확인이 필요합니다.

## 제작
선생님의 학교 / 이름
